# Ramadan Tijden Nederland

Een eenvoudige website waar je de Ramadan-gebedstijden kunt bekijken. Je kiest bovenaan je regio en de tijden worden automatisch aangepast. De keuze wordt onthouden in je browser, zodat je dit niet elke keer opnieuw hoeft te doen. Ook zie je een teller die laat zien hoeveel tijd er nog is tot het volgende gebed.

## Wat kun je op de site doen?

- Regio kiezen (Nederlandse steden)
- Tijden bekijken voor vandaag
- Afteltimer tot het volgende gebed
- Complete tabel met alle tijden voor de gekozen regio
